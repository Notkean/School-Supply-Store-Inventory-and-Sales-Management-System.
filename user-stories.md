# User Stories

## Inventory Staff
As an inventory staff, I want to add school supplies so that inventory records remain updated.

As an inventory staff, I want to update stock quantities so that inventory data remains accurate.

## Cashier
As a cashier, I want to record sales transactions so that daily sales can be tracked.

## Manager
As a manager, I want to view inventory reports so that I can monitor stock levels and sales performance.

## Administrator
As an administrator, I want to manage user accounts so that authorized users can access the system.
